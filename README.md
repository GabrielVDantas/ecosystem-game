# 🌍 Ecosystem Simulator

### 📌 About
This is a **terminal-based game** that simulates a dynamic ecosystem. The player provides initial inputs, which the system processes to define and generate the entire environment.

### ⚙️ Parameters

* **Width** `[Width]` - `[wd]` - `[Mandatory]` - `[10 - 15 - 20 - 25 - 50 - 100]`
    > Defines the horizontal size of the map.
    
* **Height** `[Height]` - `[hg]` - `[Mandatory]` - `[10 - 15 - 20 - 25 - 50 - 100]`
    > Defines the vertical size of the map.

* **Generation** `[Generation]` - `[gn]` - `[Mandatory]` - `[0 <-> 1000]`
    > Defines how much generations will be simulated.
